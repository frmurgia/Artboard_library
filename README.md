
#   **Artboard**
##   **prototyping board for artist**



Artboard is a breakout board board for teensy 3.6 made for artists, designers and musicians.



*main features:*


• 12 capacitive inputs

• 8 potentiometers

• 8 Buttons

• 12 external in/out

• 8 tactile button

• 1 switch on/off

• teensy audio shield footprints

• wifi connection (ESP12S)

• 1 led RGB

• 12 in/out digital/analog/pwm

• stereo audio output

• stereo audio input

• 8 diglta rgb led

• 1 ESP-12S

## To use this library:

#include <Artboard.h> 
Artboard artboard;

##   **Class**

 artboard.touch(0-11);

 artboard.button(0-7);

 artboard.pot(0-7);
